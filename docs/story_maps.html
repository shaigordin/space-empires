🧭 Which Story Map tool should you use?

⭐ ArcGIS StoryMaps (recommended)

  • Actively maintained by Esri — the current classroom standard.
  • Free with a public ArcGIS account.
  • No-code, robust: guided map tours, interactive maps, charts, media.
  • Best for spatial analysis + narrative together.

🔗 storymaps.arcgis.com

🪶 Knight Lab StoryMap JS (lightweight alternative)

  • Free, open-source, learn in ~10 minutes.
  • Great for georeferenced historical maps (via Map Warper).
  • Still beta, limited support.

🔗 storymap.knightlab.com

Also worth a look: Felt — modern, friendly, collaborative web maps.

Tip

💡 For this course, start with ArcGIS StoryMaps. Use Knight Lab if your project centers on overlaying a single georeferenced historical map.

🗺️ Introduction to Georeferencing

What is Georeferencing? - Assigning geographic coordinates to a historical map or raster image. - Aligns old maps with modern GIS data. - Helps track spatial changes over time.

Why does it matter? - Enables historical storytelling using spatial data. - Facilitates comparisons between past and present geographies.

Tip

💡 Example: Overlaying a 17th-century map of Jerusalem onto Google Maps to analyze urban changes.

Source: Wikimedia Commons

🎭 Introduction to StoryMap JS

🔍 What is StoryMap JS?

  • A free, open-source storytelling tool by Knight Lab.
  • Allows interactive narratives on a map-based timeline.
  • Combines text, media, and geolocation.

Key Features ✔ Integrates historical & modern maps
✔ Connects text & media (images, video, audio)
✔ Creates a sequential story across different locations

Warning

⚠ Note: StoryMap JS is not a GIS tool—it’s built for storytelling, not deep spatial analysis.

Source: Knight Lab

📜 Planning Your Narrative

Before You Start, Prepare: 1️⃣ A list of historical locations
2️⃣ A timeline or sequence of events
3️⃣ Images, videos, or primary sources
4️⃣ A georeferenced historical map

Note

🎯 Example Use Case: Mapping the spread of the Roman Empire, with each slide representing a key conquest.

🔗 Step 1: Getting a Georeferenced Map

🗂️ Using Map Warper

  • Go to Map Warper
  • Upload a historical map (e.g., Ottoman Palestine)
  • Georeference it by assigning known points
  • Export Tiles URL (in format: https://mapwarper.net/maps/tile/XXXX/{z}/{x}/{y}.png)

Map Warper UI

Tip

💡 Tip: Use at least 4 control points for accurate alignment.

✨ Step 2: Creating a StoryMap

1️⃣ Visit StoryMap JS
2️⃣ Log in with Google
3️⃣ Click “Make a StoryMap”
4️⃣ Enter a title & description

Caution

⚠ If you don’t have a Google account, you can self-host StoryMap JS (GitHub repo).

🗺️ Step 3: Adding a Custom Map

🔧 Using the Tiles URL

1️⃣ Click “Options”
2️⃣ Select “Custom” map type
3️⃣ Paste your Tiles URL from Map Warper
4️⃣ Click “Apply”

📌 Example

A georeferenced Ottoman map loaded into StoryMap JS:

🌍 Outcome

  • The historical map will replace the default basemap.
  • You can zoom & pan to explore locations.

🏛️ Step 4: Adding Historical Locations

For Each Location: 1️⃣ Click “Add Slide”
2️⃣ Enter place name (or manually adjust)
3️⃣ Add text & media (images, audio, videos)
4️⃣ Use the marker tool to position precisely

Note

📝 Example: Mapping ancient trade routes, with each stop containing artifacts and historical descriptions.

✅ Publishing & Sharing

📤 How to Publish

1️⃣ Click “Publish”
2️⃣ Copy the embed link or share URL
3️⃣ Embed into a website, blog, or Quarto page

🔗 Example: StoryMap of the Byzantine Empire

⭐ Quick-start: ArcGIS StoryMaps

1️⃣ Create a free public ArcGIS account at storymaps.arcgis.com 2️⃣ Click “New story” → choose a blank story or a guided Map Tour 3️⃣ Add a cover, then build with text, media, and map blocks 4️⃣ Insert an Express Map to draw points, routes and areas directly 5️⃣ Add your own layers (CSV of coordinates, GeoJSON) via ArcGIS Online 6️⃣ Design → pick a theme → Publish and share the link

Tip

💡 Use the Map Tour builder for sequential narratives (e.g. an Assyrian campaign route or a tour of the imperial capitals).
